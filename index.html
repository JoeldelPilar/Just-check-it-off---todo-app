About

Hi, welcome to the "just check it of" project. This is a classic ToDo list with some extra features.
This was a solo project in the training "frontend developer" at the Media Institute in Stockholm, winter 2022. The web app is built using TypeScript and Tailwind, which we started using after 4 weeks in the basic JavaScript course.

Now to some features:

  • x Add your todo in the input field.
  • x Set a deadline in the date dropdown.
  • x Sort your list by all, a to z, latest(added most recently) or need attention(closest to deadline).
  • x When there is 5 days to deadline, a blue border will appear around the task.
  • x Deadline passed? The border color now turns red.
  • x Set a category/icon to your task, school, work or info
